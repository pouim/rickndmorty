# Rick and Morty 

## Available Scripts

In the project directory, you can run:

### `npm install`

then
### `npm start`

## Tech stack:

 - React + typescript
 - Redux toolkit - To handle local states.
 - React Query - To handle server states.
 - graphql-request - To fetch data from gql endpoint.
 - mui - Theming system and styling.
